# Genius Pro

Versão renovada do jogo estilo Genius (Simon Says), com layout moderno e mais possibilidades de jogadas.

## Melhorias aplicadas

- Interface refeita com painel de estatísticas, mensagem de status e controles mais claros.
- Tabuleiro expandido para **6 cores** (mais combinações e dificuldade maior).
- Controle de velocidade da sequência.
- Persistência de recorde usando `localStorage`.
- Sons gerados dinamicamente via Web Audio API (sem depender de pasta `sounds/`).

## Como rodar

Abra o `index.html` no navegador.
